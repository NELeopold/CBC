# KPI Bonus App

Веб-приложение для бизнес-расчетов: ROI, Доход, KPI сотрудников с системой авторизации и разделением ролей.

---

## Описание проекта

**KPI Bonus App** - это веб-приложение для расчета ключевых бизнес-показателей. Приложение позволяет:

- Рассчитывать окупаемость инвестиций (ROI)
- Рассчитывать доход от продаж
- Оценивать эффективность сотрудников (KPI)
- Управлять списком сотрудников (для администратора)
- Просматривать историю всех расчетов (для администратора)

Приложение имеет систему авторизации с разделением на роли: **Администратор** и **Пользователь**.

---

##  Возможности

### Только для Администратора:
- ✅ Управление сотрудниками (добавление/удаление)
- ✅ Просмотр истории всех расчетов
- ✅ Удаление истории расчетов
- ✅ Специальный режим "Управление сотрудниками"

### Только для Пользователя:
- ✅ Регистрация нового аккаунта
- ✅ Самостоятельные расчеты

---

## Авторизация

 - Вход для существующих пользователей
 - Регистрация новых пользователей (только роль "Пользователь")
 - Предустановленный администратор

### Пример:

  Имя пользователя: ivan_petrov
  Email: ivan@example.com
  Пароль: mypass123

---

## Калькулятор ROI

  - Процент окупаемости
  - Чистую прибыль
  - Коэффициент ROI
  - Прогресс-бар
  - Цветовую индикацию (зеленый - прибыль, красный - убыток)

### Пример:

  Вложения: 100 000 ₽
  Доход: 150 000 ₽
  Результат: ROI = 50% (прибыль 50 000 ₽)

---

## Калькулятор Дохода
    
  - Общую выручку
  - Количество продаж
  - Средний чек

### Пример:

  Продажи: 150 шт.
  Средний чек: 5 000 
  Результат: Доход = 750 000 

---
## Калькулятор KPI

  - Итоговый KPI (%)
  - Выполнение плана (%)
  - Взвешенный результат

### Пример:

  Факт: 85
  План: 100
  Вес: 80%
  Результат: KPI = 68% (хорошо)

### Требования
- Любой современный веб-браузер (Chrome, Firefox, Safari, Edge, Opera)
- VS Code (рекомендуется) или любой текстовый редактор
